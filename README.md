# Resume Builder

A full-stack resume builder web app built and deployed solo.

## Features

- Google OAuth login via Passport.js
- Create and edit resumes with a live preview
- Export resume as PDF using html-to-image + jsPDF
- PostgreSQL database with Prisma ORM
- Fully deployed — frontend on Vercel, backend on Render

## Tech Stack

**Frontend:** React, JavaScript  
**Backend:** Node.js, Express.js  
**Database:** PostgreSQL, Prisma ORM  
**Auth:** Google OAuth 2.0 (Passport.js)  
**Other:** html-to-image, jsPDF, Cloudinary
